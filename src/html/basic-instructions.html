Instructions for My Password Machine


Here's how to create a unique password for each website.

  1. Open My Password Machine.
  2. Enter the website name, your username for that website, and yourmaster password.
  3. Click "Generate Password."

    4. That's it! You don't need to remember or write down this password. Just regenerate it whenever you want.
