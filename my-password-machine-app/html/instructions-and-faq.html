Frequently Asked Questions


What is My Password Machine?

An app that gives you a completely different password for each website you visit.


Why is it important to use different passwords?

That way, if someone steals your password by hacking into LinkedIn, Sony, Adobe, or another website, then the hacker will only have your password for that site -- your other passwords are safe.


Have hackers really stolen passwords from those websites?

Yes.


Hm. All right, so how does My Password Machine work?

First, you'll need a "master password" which will unlock all your other passwords.

When you use My Password Machine, the master password is the only password you will need to remember. So memorize it and don't let anyone else get it!


How do I select a master password?

Choose something that you can easily remember, and others cannot easily guess.

One good option is to roll dice to get random numbers. Then use a technique such as Diceware to convert those numbers into words. Put five or six random words together, memorize them, and you'll have a secure master password.


Aren't passwords weak if they only include words? Shouldn't I add letters, numbers, and symbols?

That doesn't always make it stronger! In reality, what's in the password isn't as important than how you choose the password.

Mathematically random passwords are the most secure. Check out "Towards Better Master Passwords" on the AgileBits blog for more info on how to choose a truly secure password.


OK. So, back to My Password Machine. How do I use it?

  1. Open a website (for example, youtube.com) and click "Log in."
  2. On the login page (which might have a different address, like accounts.google.com), copy the URL (web address) from the address bar.
  3. Open www.passwordmachine.com.
  4. Paste the URL into the "Website name" field.
  5. Enter your username for the website.
  6. Enter your master password.
  7. Click "Generate website password."
  8. My Password Machine will give you a unique password for that website.
  9. Copy that password, and paste it into the "Password" field in the website's login form.

So it's like a password manager.

Not quite! Unlike an ordinary password manger, My Password Machine doesn't store your passwords. But you can retrieve your password for any website any time you want.


How is that possible?

My Password Machine applies a formula called a "hash" to the website name and user information. For example...

   Website name:  example.com
       Username:  joe-average
Master password:  randomly-selected-memorable-words

Put those together...

example.com_joe-average_randomly-selected-memorable-words

...and hash them, and you get something like:

n9/SFj7Xu5RpLDvHLC+0Dg==

And joe-average can use this as his password for example.com.

It can be re-created at any time by entering 'example.com', 'joe-average', and 'randomly-selected-memorable-words' into Password Machine. There's no need to write it down!


So I can get a strong and unique password for every username and website, without storing a list of passwords that hackers can steal.

Exactly!


What if I already have a website account and password?

Then you'll have to log in to that website and change your password there to the one that My Password Machine gives you.

But if your old password follows a well-known pattern -- such as a family member's name, or a keyboard pattern like "zxcvbnm", or a letter-symbol substitution like "My$ecretw0rd -- it's easy for hackers to guess and you should change it anyway!

My Password Machine creates passwords that don't follow a pattern, and so cannot be easily guessed.
